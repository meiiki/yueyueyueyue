1.使用ssh方式关联远程仓库
  关联远程仓库有两种:https  ssh
  而https在每次推送时,都需要输入用户名与密码,比较麻烦
  配置SSH:
    查看本机是否有密钥
      密钥包含两个:公钥,私钥
      创建生成密钥:
        ssh-keygen -t rsa -C '邮箱地址'
      关联推送

2.拉取数据
  将远程仓库的内容下载到本地
  第一种: 先进入目标文件夹,将其初始化为本地仓库
    使用命令git pull 远程仓库地址 master
  第二种: 进入文件夹,无序初始化本地仓库,使用clone命令
    git clone 远程仓库地址
    该命令下载的就是一个完整的仓库文件夹
